# Hair Salon Booking System
A hair salon booking application developed with HTML, CSS, JavaScript, Node.js, Express.js and SQLite.

## Installation
1. Clone the repository:  

```bash 
git clone https://github.com/katerinamar28/Hair-Salon-Booking
```

2. Install NPM packages:  

```bash
npm install
```

3. Start the server:

```bash
node server.js
```

4. Open in browser:  

Reservation form: http://localhost:3000     

Reservation overview table: http://localhost:3000/reservations.html
